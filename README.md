# 生物重要単語集
### 環境
- 生物的環境
- 非生物的環境(光・水・土壌・大気・温度)
### 作用-(非生物的環境—>生物的環境)
- 光・水・二酸化炭素・土壌中の栄養塩類が樹木の成長に影響すること
### 環境形成作用(生物的環境—>非生物的環境)
- 樹木の成長によって低木層などが暗くなること
### 適応
- 性質が環境中で生活しやすいよう都合良くなっていること
### 生活形
- 環境への適応を反映した形態
### 植物
- 一年生植物
    - 種子が発芽してから一年以内に結実して枯死する植物
- 多年生植物
    - 地下部などに栄養分を貯蔵しながら一年を超えて生育する植物
### 木本(もくほん)
- 低木
- 亜高木
- 高木
+ 落葉樹
    - 葉を落とさない木
+ 常緑樹
    - 雨季や乾季に葉を落とす木
- 針葉樹
- 広葉樹
